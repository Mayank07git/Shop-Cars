# 🚗 Shop-Cars: Premium Car Marketplace Clone

A high-performance, fully responsive car marketplace web application built using **Next.js** and **Vanilla CSS Modules**. This project demonstrates intermediate-level frontend architecture, dynamic routing, and advanced UI/UX patterns.

## 🚀 Live Demo
https://shop-cars-nu.vercel.app/

## ✨ Key Features
- **Dynamic Routing:** Utilizes Next.js `[id].js` routes for automated product pages.
- **Advanced Filtering:** Multi-category checkbox filtering (Make, Year, Budget, Transmission, etc.) with complex logic.
- **Real-time Search:** Instant live-search functionality on the homepage.
- **Mobile-First UX:** 
  - Slide-in Left-to-Right Hamburger Menu.
  - Bottom-to-Top sliding Filter Drawer on mobile.
  - Interactive "Book Test Drive" Modal with date/time selectors.
- **Premium Aesthetics:** Hand-crafted CSS Module system with smooth gradients, shimmer skeleton loaders, and micro-animations.

## 🛠️ Tech Stack
- **Framework:** Next.js (Pages Router)
- **Styling:** Vanilla CSS Modules (Scoped)
- **Backend:** Next.js API Routes
- **Database:** Local JSON Mock DB
